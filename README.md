# 🏎️ Ford GT MK IV

Sitio web institucional desarrollado como proyecto final para CoderHouse, dedicado a mostrar la historia, evolución y especificaciones técnicas del Ford GT MK IV mediante una interfaz moderna, responsive y optimizada para SEO.

---

## 🌐 Demo

👉 https://TU-USUARIO.github.io/FORD/

---

## 📌 Características

- Diseño responsive adaptado a escritorio, tablet y móvil.
- Desarrollo con HTML5 semántico.
- Estilos organizados mediante SASS.
- Componentes personalizados con Bootstrap 5.
- Optimización SEO básica.
- Interactividad con JavaScript.
- Almacenamiento de datos mediante LocalStorage.
- Animaciones y transiciones CSS.

Desarrollar un sitio web institucional aplicando buenas prácticas de desarrollo frontend, estructura semántica, optimización SEO, diseño responsive e integración de JavaScript para mejorar la experiencia del usuario.
